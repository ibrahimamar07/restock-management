## 📖 Overview

**Restock Management** adalah sistem manajemen restock yang dirancang untuk membantu bisnis dalam mengotomatisasi proses pengisian ulang stok barang.  
Dengan sistem ini, pengguna dapat memantau ketersediaan produk secara real-time, menerima notifikasi ketika stok mencapai batas minimum, serta mendapatkan rekomendasi pemesanan cerdas berdasarkan data historis penjualan.

Proyek ini ditujukan untuk meminimalkan risiko _stockout_ dan _overstock_, mengurangi beban operasional, dan meningkatkan efisiensi dalam pengambilan keputusan terkait manajemen inventaris.

---

## 🎯 Goals & Benefits

### 🎯 Tujuan

Mengembangkan perangkat lunak yang mampu:

-   Mengotomatisasi pemantauan level stok.
-   Memberikan notifikasi stok minimum.
-   Menyediakan rekomendasi restock yang cerdas.
-   Meningkatkan efisiensi operasional dan ketepatan pengambilan keputusan.

### 💡 Manfaat

-   Menghindari _stockout_ dan _overstock_.
-   Mempercepat proses pemesanan ulang barang.
-   Menyediakan data akurat untuk analisis dan strategi bisnis.
-   Mengurangi risiko kesalahan manusia (_human error_).

## ⚙️ Key Features

| Fitur                                   | Deskripsi                                                                                                          |
| --------------------------------------- | ------------------------------------------------------------------------------------------------------------------ |
| **📊 Dashboard Monitoring Stok**        | Menampilkan kondisi stok barang secara visual dan real-time, termasuk produk yang hampir habis dan tren penjualan. |
| **📦 Manajemen Data Produk**            | Mengelola data produk seperti SKU, nama, kategori, harga beli, dan pemasok.                                        |
| **🔁 Pemantauan Stok Real-Time**        | Stok otomatis ter-update setiap transaksi penjualan atau penerimaan barang.                                        |
| **⚠️ Notifikasi Stok Minimum**          | Memberikan alert otomatis ketika stok mencapai batas minimum.                                                      |
| **🧠 Rekomendasi Pemesanan Cerdas**     | Sistem menganalisis data historis penjualan untuk merekomendasikan produk dan jumlah restock optimal.              |
| **🧾 Manajemen Pesanan Pembelian (PO)** | Membuat, mengirim, dan melacak status _purchase order_ ke pemasok langsung dari sistem.                            |
| **🏭 Manajemen Pemasok**                | Menyimpan data pemasok (kontak, alamat, produk yang disuplai).                                                     |
| **📈 Laporan & Analitik**               | Menyediakan laporan nilai inventaris, rotasi stok, dan produk terlaris untuk mendukung keputusan bisnis.           |
